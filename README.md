# StarBringer
- 스타브링어는 3D RPG Game이며, 홀로 기획 & 개발 & 디자인을 통해 구현한 첫 게임입니다.
- Unity3D, C#을 통해 개발되었습니다.
- 이 repo에는 스크립트에 해당하는 파일만 존재합니다.
